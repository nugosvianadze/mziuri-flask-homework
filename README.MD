## 🧠 Backend (main.py)

### 1. Flask app

``` python
from flask import Flask, render_template

app = Flask(__name__)
```

### 2. Data (სიმულაციის მაგალითი, დაამატეთ, შეცვალეთ)

``` python
products = [
    {"id": 1, "name": "Laptop", "price": 1200},
    {"id": 2, "name": "Phone", "price": 800},
    {"id": 3, "name": "Tablet", "price": 600},
]
```

### 3. Route `/`

``` python
@app.route("/")
def index():
    """ გვიბრუნებს თიმფლეითს რომელიც არენდერებს ყველა პროდუქტს"""
```

### 4. API endpoint `/api/products`

``` python
@app.route("/api/products")
@app.route("/api/products/<search>)
def get_products.........
"""ამ ეიპიაი ენდფოინთმა უნდა დააბრუნოს ყველა პროდუქტის ლისტი თუ search ქივორდი არ გადავეცით. თუ გადავეციტ მისამართშლი /api/products/phon -> უნდა დამიბრუნოს ისეთ პროდუქტების სია რომელთა სახელშიც შედის 'phon' -> exact match არ გვინდა, გვინდა დამთხვევის ლოგიკა."""
```


### 5. Run app

``` python
if __name__ == "__main__":
    app.run(debug=True)
```
